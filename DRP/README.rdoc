== README

Distributed Route Planning Edmonton

* Ruby version: 2.2.1
* Rails version: 4.2.5.1

Installation Instructions:

Install RVM: https://rvm.io/rvm/install

Use the correct Ruby Verion: rvm use 2.2.1 (This will install thsi version if you dont have it)

Create a gemset: rvm gemset create nameofgemset

Select the gemset: rvm use 2.2.1@nameofgemset

Install all the gems you need: bundle install

run the server: rails s



<tt>rake doc:app</tt>.
